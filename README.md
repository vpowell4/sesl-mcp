## Getting Started

### Prerequisites
- Git
- Python 3.10+

### Clone
```bash
git clone https://github.com/vpowell4/sesl-mcp.git
cd sesl-mcp
```

### Run
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt  # if present
python <entrypoint>.py
```
